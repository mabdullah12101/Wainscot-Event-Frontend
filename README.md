## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/mabdullah12101/Wainscot-Event-Frontend.git
   ```
2. Add .env file at root folder project
   ```sh
   REACT_APP_CLOUDINARY_URL_IMAGE = "Your Cloudinary URL"

   REACT_APP_CLOUDINARY_DEFAULT_IMAGE = "Your Image Default"
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run project
   ```sh
   npm start
   ```
